# Design Plan for Venkatakrishnan Asuri's Personal Website

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a2332) - Professional, academic sophistication
- **Secondary**: Warm Gold (#d4af37) - Achievement, excellence, IIT heritage
- **Accent**: Sage Green (#87a96b) - Growth, learning, balance
- **Background**: Off-white (#fafafa) - Clean, readable, academic
- **Text**: Charcoal (#2c3e50) - High contrast, professional readability

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings, academic gravitas
- **Body Font**: "Inter" - Clean, modern sans-serif for excellent readability
- **Code Font**: "JetBrains Mono" - For technical content and achievements

### Visual Language
- Academic excellence with modern sophistication
- Clean, structured layouts reflecting analytical thinking
- Subtle geometric patterns suggesting mathematical precision
- Professional portfolio aesthetic suitable for graduate school applications

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth transitions and micro-interactions
- **Typed.js**: Typewriter effect for key phrases
- **Splitting.js**: Text animation effects for headings
- **ECharts.js**: Data visualization for academic achievements
- **Splide.js**: Image carousels for projects/achievements

### Effects & Animation
- **Hero Section**: Typewriter animation for name and tagline
- **Section Transitions**: Subtle fade-in animations on scroll
- **Achievement Cards**: Hover effects with 3D tilt
- **Language Proficiency**: Animated progress bars
- **Timeline**: Interactive education/work history

### Styling Approach
- **Layout**: Grid-based, responsive design
- **Cards**: Subtle shadows, rounded corners, hover animations
- **Navigation**: Fixed header with smooth scrolling
- **Background**: Minimal geometric patterns, low opacity
- **Interactive Elements**: Smooth hover states, micro-interactions

### Header Effect
- Clean navigation bar with subtle background blur
- Smooth color transitions on scroll
- Logo/name with animated underline on hover

### Content Sections
1. **Hero**: Personal introduction with animated text
2. **Education**: Interactive timeline with achievements
3. **Experience**: Work history with detailed descriptions
4. **Skills**: Visual representation of languages and competencies
5. **Achievements**: Academic accomplishments with data visualization
6. **Interests**: Personal pursuits with engaging visuals
7. **Contact**: Professional contact information

### Responsive Design
- Mobile-first approach
- Flexible grid layouts
- Optimized typography scaling
- Touch-friendly interactive elements
