# Neon Dark Theme Design Plan for Venkatakrishnan Asuri's Website

## Design Philosophy

### Color Palette
- **Background**: Deep Black (#0a0a0a) - Neon cyberpunk aesthetic
- **Primary Neon**: Electric Blue (#00d4ff) - Main accents and highlights
- **Secondary Neon**: Neon Green (#39ff14) - Success states and achievements
- **Accent Neon**: Neon Yellow (#ffff00) - Warning and attention elements
- **Text Primary**: White (#ffffff) - High contrast readability
- **Text Secondary**: Light Gray (#e0e0e0) - Secondary information
- **Card Background**: Dark Gray (#1a1a1a) - Content containers
- **Border Color**: Neon Blue with opacity (#00d4ff20) - Subtle neon borders

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings with neon glow
- **Body Font**: "Inter" - Clean, modern sans-serif for excellent readability
- **Code Font**: "JetBrains Mono" - For technical content and achievements
- **Neon Glow Effect**: Text-shadow for neon lighting effect

### Visual Language
- Cyberpunk neon aesthetic with dark backgrounds
- Glowing borders and subtle neon accents
- Compact, information-dense layout
- High contrast for accessibility
- Modern, tech-forward design suitable for academic/professional use

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth transitions and micro-interactions with neon glow effects
- **Typed.js**: Typewriter effect with neon text color
- **Splitting.js**: Text animation effects with neon glow
- **ECharts.js**: Data visualization with neon color scheme
- **Splide.js**: Image carousels with neon styling

### Effects & Animation
- **Hero Section**: Typewriter animation with neon blue text
- **Section Transitions**: Fade-in animations with neon glow
- **Achievement Cards**: Hover effects with 3D tilt and neon borders
- **Language Proficiency**: Animated neon progress bars
- **Timeline**: Interactive education/work history with neon accents

### Styling Approach
- **Layout**: Grid-based, responsive design with compact sections
- **Cards**: Dark backgrounds with neon borders and glow effects
- **Navigation**: Fixed header with neon accents and smooth scrolling
- **Background**: Pure black with subtle geometric neon patterns
- **Interactive Elements**: Neon hover states and micro-interactions

### Header Effect
- Clean navigation bar with neon blue accents
- Smooth color transitions on scroll
- Logo/name with animated neon underline on hover

### Content Sections (Compact Design)
1. **Hero**: Personal introduction with animated neon text
2. **Education**: Interactive timeline with key achievements
3. **Experience**: Work history with detailed descriptions
4. **Skills**: Visual representation with neon progress bars
5. **Achievements**: Academic accomplishments with neon styling
6. **Projects**: Featured projects with neon card design
7. **Contact**: Professional contact information

### Responsive Design
- Mobile-first approach
- Flexible grid layouts
- Optimized typography scaling
- Touch-friendly interactive elements
- Neon effects adapted for mobile screens
